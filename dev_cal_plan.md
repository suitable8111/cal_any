역할: 당신은 Next.js(App Router), TypeScript, Tailwind CSS, 그리고 Shadcn UI에 정통한 시니어 프론트엔드 개발자이자 UI/UX 디자이너입니다. SEO(검색엔진 최적화) 최적화와 카카오 애드핏(Kakao AdFit) 광고 배너 배치를 고려한 '만능 계산기 및 정보 웹사이트'를 구축하려고 합니다.

[프로젝트 개요]
- 배포 플랫폼: Vercel (Fast, Serverless, Edge Network 최적화)
- 주요 기능: 다양한 생활 밀착형 계산기 기능 및 관련 정보 탭 제공
- 수익화 모델: 카카오 애드핏 광고 탑재 (상단, 하단, 계산 결과창 주변 등)
- 핵심 가치: 직관적인 UI, 빠른 계산 속도, 모바일 최적화(Responsive), 검색 노출을 위한 SEO 구조

[구현할 주요 페이지 및 기능]
1. 메인 페이지 (Home): 
   - 전체 계산기 리스트를 깔끔한 그리드(Grid) 형태의 카드로 나열.
   - 사용자가 원하는 계산기를 빠르게 찾을 수 있는 '실시간 검색 바' 포함.
2. 만 나이 계산기 (Age Calculator):
   - 생년월일 및 기준일 입력 -> 만 나이, 띠, 연 나이, 다음 생일 일수 계산.
3. 부동산 세금 계산기 (Real Estate Tax):
   - 주택 보유 수, 취득가액, 조정대상지역 여부 입력 -> 취득세, 지방교육세, 농어촌특별세 자동 계산 및 요율표 정보 제공.
4. 자동차 벌금/과태료 조회 및 계산기 (Traffic Fine):
   - 위반 항목(과속, 신호위반 등), 차종(승용/승합), 보호구역 여부 선택 -> 벌점 및 과태료/범칙금 결과 출력.
5. 전기차 요금 계산기 (EV Charging Cost):
   - 차량 배터리 용량, 현재 잔량, 목표 충전량, 충전기 종류(급속/완속), 계절/시간대 선택 -> 예상 충전 시간 및 비용 계산.

[광고 배너 (카카오 애드핏) 배치 요구사항]
- 사용자의 흐름을 방해하지 않으면서 클릭률(CTR)을 높일 수 있는 컴포넌트(`KakaotrackAd.tsx`)를 설계해 주세요.
- 배치 위치:
  1. 헤더 바로 아래 (가로형 배너)
  2. 계산기 입력 폼과 결과 창 사이 (콘텐츠 삽입형 배너)
  3. 사이드바 또는 푸터 위 (레이아웃 최적화)
- Next.js의 `Script` 컴포넌트를 활용하여 카카오 애드핏 스크립트가 비동기로 안전하게 로드되도록 코드를 작성해 주세요.

[기술 스택 및 코드 작성 스타일]
- Framework: Next.js (최신 App Router 기반)
- Language: TypeScript (엄격한 타입 지정)
- Styling: Tailwind CSS (모던하고 깔끔한 파스텔 톤 또는 시인성 높은 다크/라이트 모드 지원)
- Components: 깔끔하고 재사용 가능한 컴포넌트 기반 구조화 (`/components`, `/app` 폴더 구조 정의)

3. 메인 레이아웃(`layout.tsx`)과 메인 페이지(`page.tsx`) 코드를 구현해 주세요.
